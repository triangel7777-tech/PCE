# External Run Capture Packet v2.0

Required:

- Repository
- Workflow name
- Workflow URL
- Commit SHA
- Runner
- Actor/reviewer
- Independent from builder: true/false
- Artifact name
- Artifact SHA-256
- Completed verifier report path
- Attachment manifest path
- Command transcript paths
- Human reviewer decision
- Promotion recommendation

Boundary: This packet becomes evidence only when filled from a real external run.
